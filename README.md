
KD Ltd Website Mockup (Static HTML)
==================================

What you get
------------
A complete, professional-style contractor website mockup that you can:
- Host anywhere as static HTML
- Use as a blueprint to rebuild inside Wix (recommended)
- Hand to a designer/developer as a reference

Pages included
--------------
- index.html (Home)
- services.html + /services/* (service detail pages)
- projects.html + /projects/* (project detail pages)
- before-after.html
- about.html
- reviews.html
- blog.html + /blog/* (sample posts + project documentation template)
- contact.html

How to run locally
------------------
Option A: open index.html in your browser.
Option B: serve locally (recommended) using Python:

  python -m http.server 8000

Then open: http://localhost:8000

How to adapt this for Wix
-------------------------
- Create the same pages in Wix using the sitemap.
- Copy the section order from index.html:
  Hero -> Services -> Projects -> Why Choose -> Process -> Before/After -> Testimonials -> CTA -> Contact
- Replace SVG placeholders in /assets/img with real project photos.
- Use the contact form fields as your Wix form structure.

Content checklist (high impact)
-------------------------------
1) Replace hero headline + add CTAs
2) Add 10 project pages (before/during/after)
3) Add Google Reviews (best trust signal)
4) Add individual service pages (SEO)
5) Add service area list (local SEO)

Notes
-----
- All images are SVG placeholders so the site is fast and self-contained.
- Colors and spacing are intentionally premium/minimal.
