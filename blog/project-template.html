Project documentation template

Use this for every job so your website portfolio grows automatically over time.

1) Basic info

  • Project title: (e.g., Custom Media Wall)
  • Location: City/Area
  • Completion date: Month/Year
  • Service category: Custom cabinetry / Kitchen / Renovation / Repairs

2) Goal + story

  • Client goal: What problem did we solve?
  • Solution: What did we build?
  • Outcome: What changed? Storage? Look? Function?

3) Photos (shot list)

  • Before: wide + close-up of problem area
  • During: measuring, assembly, installation
  • After: wide + 3 close-ups (joinery, finish, hardware)

4) Details (optional but powerful)

  • Materials: plywood/solid wood/trim/hardware
  • Timeline: days/weeks
  • Budget range: (if you want transparency)
  • Testimonial: 1–2 sentences tied to the project
Publish format: Turn each completed template into a project page with before/after images and a short story.
